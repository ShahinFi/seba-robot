# SEBA-BOT

SEBA-BOT is a two-wheeled self-balancing robotic platform for control, autonomy, and real-time robotics development.

## Overview

The project originates from inverted pendulum balancing and state-space control, and is designed to evolve into a broader robotic system supporting sensing, estimation, and autonomous behavior.

## Current Work

* dynamics modeling
* state-space formulation
* velocity control
* augmented LQR design
* simulation and validation

## Planned Extensions

* localization
* mapping
* sensor fusion
* UWB-based positioning
* autonomous navigation
* AI/ML-based control and decision systems

## Repository Structure

* `control/` — dynamics, modeling, and control design
* `hardware/` — mechanical and electronics design
* `src/` — embedded and software implementation
* `docs/` — additional documentation

## Notes

This project is under active development, and the architecture is intended to remain open to future control, software, and hardware extensions.
